How to install

bower install yiting007/angular-dUtil --save
This will copy the source code files into your bower_components folder, along with its dependencies. Next add the script file into your application. 
<script src="bower_components/angular-dUtil/dUtil.js"></script>
The css file directory is 
bower_components/angular-dUtil/dUtil.css
Modify this file if needed then add it into your application too.

How to use

Add the 'dUtilApp' module as a dependency to your application module. 
var myAppModule = angular.module('MyApp', ['dUtilApp']);
