Play with Arduino UNO using Golang (and Lego!)

Requirements

  • Having Golang installed and GOPATH set;
  • Having an arduino UNO with some other components;
  • It will be more fun if you also have some Lego parts.
  1. Install Homebrew
    2. Because I'm using Mac, and Gort for Mac uses Homebrew.
  2. Install Gort, then
    3. $ ./gort scan serial
    $ ./gort arduino install
    $ ./gort arduino upload firmate [serial]
  3. Install tarm/goerial
    4. It has been replaced by tarm/serial but Gobot still uses the old one. 
    $ go get github.com/tarm/goserial
  4. Install Gobot for arduino
    5. Then you should be able to run the examples using Gobot!
