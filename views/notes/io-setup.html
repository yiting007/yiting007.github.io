Steps to set up a github io page using AngularJS

  1. Create a git repo named username.github.io.
  2. Clone the io repo to local, create a src branch.
  3. Install yoeman angular generator: https://github.com/yeoman/generator-angular
  4. In the git repo (using Yeoman) $> yo angular to create an angularjs project.
  5. Install load-grunt-tasks https://github.com/sindresorhus/load-grunt-tasks
  6. In Gruntfile.js, add: require('load-grunt-tasks')(grunt);
  7. Use load-build-control https://github.com/robwierzbowski/grunt-build-control
  8. In Gruntfile.js, add:
  9. Register deploy task
  10. $> grunt build (Get the dist folder)
  11. $> grunt serve (Test locally, serve task should be in Gruntfile.js by default)
  12. $> grunt deploy (Deploy dist to master branch)
